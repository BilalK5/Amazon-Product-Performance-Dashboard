📦 Amazon Product Performance & Customer Engagement Dashboard

 📌 Project Overview

This project analyzes Amazon product data to identify trends in customer engagement, pricing behavior, product performance, and shipment patterns. An interactive Power BI dashboard was developed to transform raw data into actionable business insights.

The dashboard helps answer questions such as:

- Which product categories generate the most engagement?
- How does pricing influence review volume?
- Which products perform best?
- Are there noticeable order trends over time?
- How consistent are shipment patterns across categories?

---

## 🎯 Objectives

The project aims to:

✔ Analyze customer engagement through reviews  
✔ Identify top-performing product categories  
✔ Examine pricing patterns across products  
✔ Explore shipment and order trends  
✔ Build an interactive dashboard for business decision-making

---

## 🛠 Tools Used

- Power BI
- DAX
- Excel
- Data Cleaning
- Data Visualization
- KPI Design

---

## 📈 Key KPIs

The dashboard tracks:

- Total Reviews
- Average Price
- Total Products
- Average Reviews per Product

These KPIs provide an executive summary of product performance.

---

## 🔍 Dashboard Components

### 1. Product Category Performance
Shows customer engagement by category.

### 2. Price vs Reviews Analysis
Examines relationships between pricing and engagement.

### 3. Top Products Table
Identifies products with highest review activity.

### 4. Shipment Analysis
Explores fulfillment consistency.

### 5. Order Trends
Tracks activity over time.

---

## 💡 Key Insights

Examples:

- Certain product categories dominate review volume.
- Lower-priced products tend to receive stronger engagement.
- Shipment patterns are highly standardized.
- Customer activity shows temporal variation.
